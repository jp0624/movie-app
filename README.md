# MovieApp

A modern React + TypeScript movie browsing application powered by the TMDB API.

## Features

- Infinite scroll
- Debounced search
- Tabs for Popular, Top Rated, Trending, Now Playing
- Favorites with sorting and ratings
- Light/dark mode
- Movie details page
- Tailwind-only UI
- Modern folder structure

## Getting Started

1. Install dependencies:

```bash
npm install
```

2. Create `.env` in project root:

```
VITE_TMDB_KEY=REPLACE_WITH_YOUR_TMDB_V4_TOKEN
```

3. Start development server:

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Preview

```bash
npm run preview
```
